# NLP Diagrammer Specification

## Heuristics

### Priority/Constraint Logic
- Keywords: prioritizes, ensures, provides, offers, focuses, without, in
- Template: Stacked or directional flow emphasizing the priority.

### Compare Logic
- Keywords: vs, versus, whereas, while, but
- Template: Side-by-side comparison or split view.

### Category Logic
- Fallback for noun-verb-noun structures.

## Glue Vocabulary
- Allowed: 'Priority:', 'Constraints:', '|', '->', ',', ':'
